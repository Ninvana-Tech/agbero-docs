# Create Merchant

Create a merchant account in the system.

## Endpoint

**POST** `/api/v1/merchant/create`

## Request Body

```json
{
  "email": "odunmilade@gmail.com",
  "password": "whiteman@5",
  "name": "Hody INC",
  "contact": "08089040350",
  "address": "Test Address"
}
```
